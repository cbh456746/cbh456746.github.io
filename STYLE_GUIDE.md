# Style guide

- Prefer a useful title and a one-sentence `subtitle`.
- Write Korean primarily; retain original quotations and source language where useful.
- Explain an unfamiliar abbreviation on first use.
- Use headings in order: `##`, then `###`.
- Use alt text that describes the point of an image, not only its file name.
- Do not put passwords, private keys, personal phone numbers, or unreviewed personal data into the repository.
